# Mobile Market Insights – Power BI Dashboard

## Overview

**Mobile Market Insights** is an interactive **Power BI dashboard** developed to analyze mobile phone sales performance across different regions, brands, and time periods. The dashboard converts raw sales data into meaningful visual insights, helping users understand revenue trends, product performance, and market demand.

## Features

* Interactive sales performance visualizations
* Brand-wise and region-wise sales analysis
* Monthly and yearly revenue trend tracking
* KPI indicators for total sales, revenue, and quantity sold
* Dynamic filters and slicers for detailed exploration

## Tools & Technologies

* **Tool:** Power BI Desktop
* **Data Source:** Excel / CSV / SQL
* **Data Processing:** Power Query Editor
* **Visualization:** Charts, KPI Cards, and Slicers

## Usage

1. Download the `.pbix` file from the repository.
2. Open the file using **Power BI Desktop**.
3. Refresh the dataset if required.
4. Use filters and slicers to explore interactive insights.

## Applications

* Sales trend analysis
* Brand performance comparison
* Market demand analysis
* Data-driven business decision-making
